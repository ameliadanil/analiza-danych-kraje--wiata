# Analiza danych o krajach świata

**Przedmiot:** Zarządzanie Big Data

**Autor:** Amelia Danił
**Nr albumu:** 159803

## Opis projektu

Projekt polega na zbudowaniu prostego pipeline’u danych: pobranie danych o krajach świata z API REST Countries, ich przetworzenie do formatu tabelarycznego (DataFrame), zapis do bazy SQLite, wykonanie analiz przy użyciu zapytań SQL oraz wizualizacja wyników w postaci wykresu.
